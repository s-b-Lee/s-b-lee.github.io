개인정보처리방침 (Privacy Policy)

시행일: 2026-02-09
서비스명: 🫧이미지 레시피 - 직접 설계하는 내 이미지
회사/퍼블리셔: s-b-lee
문의: sargasght@gmail.com
본 개인정보처리방침은 “🫧이미지 레시피”(이하 “서비스”)가 이용자의 정보를 어떤 방식으로 수집·이용·보관·제공하는지 설명합니다.

1. 수집하는 정보

(1) 이용자가 직접 입력하는 정보

  • 스타일 키워드 선택값(예: 세련됨, 우아함 등)
  • 추가 정보 입력값(좋아하는 스타일/싫은 요소/현실 제약 등)
  • 추구미 챗봇 대화 내용(사용자 질문/메시지)

(2) 이용자가 업로드하는 이미지(선택)

  • 이용자가 이미지 분석 기능을 사용할 경우, 업로드 이미지가 분위기/스타일 분석 요청을 위해 처리될 수 있습니다.

(3) API 키/토큰(사용자 입력값)

  • OpenAI API Key
  • Pinterest Access Token(Bearer, OAuth 기반)

중요: 서비스는 Pinterest 계정 비밀번호를 수집하지 않습니다. Pinterest 기능은 Access Token으로만 동작합니다.

2. 이용 목적

  1. 추구미 리포트/가이드 생성(OpenAI 텍스트 생성)
  2. 업로드 이미지 분위기 분석(선택, OpenAI 비전 요청)
  3. Pinterest 참고 이미지(핀) 검색/표시(선택, Pinterest API 호출)
  4. 예시 코디 시각화 이미지 생성(선택, OpenAI 이미지 생성)
  5. 추구미 챗봇 응답 생성(OpenAI 텍스트 생성)

3. 보관 및 파기

  • 서비스는 입력값/결과를 주로 앱 세션(Streamlit session_state)에서 표시하며, 초기화 또는 세션 종료 시 삭제될 수 있습니다.
  • Pinterest에서 조회된 핀 정보는 화면 표시 목적이며, 별도 데이터베이스에 영구 저장하지 않습니다(세션 캐시 수준).

4. 제3자 제공 및 처리위탁(외부 API)

(1) OpenAI

  • 목적: 리포트/챗봇 응답 생성, 이미지 분석(선택), 코디 시각화(선택)
  • 전송될 수 있는 정보: 사용자가 입력한 텍스트, 업로드 이미지(선택), 생성 프롬프트

(2) Pinterest

  • 목적: 참고 이미지(핀) 검색/표시
  • 전송될 수 있는 정보: 검색어(키워드 기반), Access Token

5. 이용자의 권리

  • 입력/업로드 중단: 키/토큰 입력 및 이미지 업로드를 언제든지 중단할 수 있습니다.
  • 세션 데이터 삭제: 앱 내 초기화 기능을 통해 세션에 저장된 정보를 삭제할 수 있습니다.
  • 문의: 개인정보 관련 문의는 sargasght@gmail.com 으로 연락할 수 있습니다.

6. 보안

  • HTTPS 제공(배포 설정에 따라)
  • 필요 최소 범위 내 정보 처리
  • 키/토큰은 사용자 입력 기반이며 불필요한 노출을 최소화

7. 아동(만 13세 미만)

서비스는 만 13세 미만 아동을 대상으로 설계되지 않았습니다.

8. 정책 변경

본 방침이 변경되는 경우, 페이지 업데이트로 안내합니다.
