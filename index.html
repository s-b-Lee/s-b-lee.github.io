개인정보처리방침 (Privacy Policy)

본 개인정보처리방침은 “🫧이미지 레시피 - 직접 설계하는 내 이미지”(이하 “서비스”)가 이용자의 개인정보를 어떤 방식으로 수집·이용·보관·제공하는지 설명합니다.

1. 수집하는 정보

서비스는 이용자가 입력하거나 업로드하는 정보를 아래와 같이 처리할 수 있습니다.

(1) 이용자가 직접 입력하는 정보

  • 스타일 키워드 선택값 (예: 세련됨, 우아함 등)
  • 추가 정보 입력값 (좋아하는 스타일, 싫은 느낌, 현실 제약/조건 등)
  • 추구미 챗봇 대화 내용 (사용자 질문/메시지)

(2) 이용자가 업로드하는 이미지

  • 이용자가 “(선택) 이미지 업로드 — 추구미 분위기 분석” 기능을 사용하면, 업로드한 이미지가 분위기/스타일 분석 요청을 위해 처리될 수 있습니다.

(3) API 키/토큰(사용자 입력값)

  • OpenAI API Key
  • Pinterest Access Token (Bearer)

중요: 서비스는 Pinterest 계정 비밀번호를 수집하지 않습니다. Pinterest 기능은 사용자가 발급받은 Access Token(OAuth 기반)으로만 동작합니다.

2. 정보의 이용 목적

서비스는 수집된 정보를 다음 목적에 한해 이용합니다.

  1. 추구미 리포트 생성 — 키워드/추가 입력/이미지 분석 결과(선택)를 바탕으로 텍스트 리포트 생성
  2. 이미지 분위기 분석(선택 기능) — 업로드 이미지를 기반으로 키워드 추정/근거/주의 문구 생성
  3. Pinterest 참고 이미지 표시(선택 기능) — Pinterest API 호출을 통해 참고 이미지(핀) 표시
  4. 예시 코디 시각화(선택 기능) — OpenAI 이미지 생성 API를 통해 코디 시각화 생성
  5. 추구미 챗봇 응답 생성 — 사용자의 질문과 컨텍스트 기반으로 코칭 답변 생성

3. 보관 및 파기

(1) 서비스 내 저장 방식

  • 서비스는 사용자의 입력값(키워드, 텍스트, 챗봇 대화, 리포트 결과 등)을 Streamlit 세션 상태(session_state)에 저장하여 화면에 표시할 수 있습니다.
  • 이는 일반적으로 브라우저 세션/앱 실행 동안 유지되며, 사용자가 “초기화” 버튼을 누르면 세션 내 기록을 삭제합니다.

(2) 업로드 이미지의 보관

  • 업로드 이미지는 분석 요청을 위해 처리될 수 있으며, 서비스는 기본적으로 앱 내부 표시 및 분석을 위해 일시적으로 취급될 수 있습니다.
  • 사용자가 초기화/세션 종료 시 세션 상태에 저장된 데이터는 삭제됩니다.

(3) OpenAI API Key / Pinterest Access Token 보관

  • 이용자가 입력한 키/토큰은 앱 실행 중 요청 수행을 위해 사용됩니다.
  • 서비스는 이를 별도의 데이터베이스에 저장하지 않습니다.
  • 사용자는 키/토큰 입력을 중단하거나 초기화하여 언제든지 제거할 수 있습니다.

4. 제3자 제공 및 처리위탁(외부 API 사용)

서비스는 기능 제공을 위해 아래 외부 서비스(API)를 호출할 수 있습니다.

(1) OpenAI (텍스트/이미지 생성)

  • 이용 목적: 추구미 리포트 생성, 챗봇 응답 생성, 예시 코디 시각화(이미지 생성)
  • 전송될 수 있는 정보: 이용자가 입력한 텍스트(키워드/추가 정보/질문), 업로드 이미지(선택 시), 생성 프롬프트
  • OpenAI API Key는 이용자가 직접 입력한 값을 사용합니다.

(2) Pinterest (참고 이미지 검색)

  • 이용 목적: Pinterest 참고 이미지(핀) 조회 및 표시
  • 전송될 수 있는 정보: 이용자가 입력한 검색어, Access Token
  • Pinterest Access Token은 이용자가 직접 입력한 값을 사용합니다.

서비스는 이용자 동의 없이 제3자에게 이용자의 개인정보를 판매하지 않습니다.
외부 API 제공자(OpenAI, Pinterest)의 처리 방식은 각 제공자의 정책을 따릅니다.

5. 이용자의 권리

  • 입력/업로드 중단: 키/토큰 입력 및 이미지 업로드를 중단할 수 있습니다.
  • 세션 데이터 삭제: 서비스 내 “초기화” 버튼을 통해 세션 상태에 저장된 데이터를 삭제할 수 있습니다.
  • 문의: 개인정보 관련 문의는 상단 문의처(이메일)로 요청할 수 있습니다.

6. 보안 조치

  • HTTPS 환경에서의 서비스 제공(배포 설정에 따라 적용)
  • API Key/Access Token은 사용자 입력 기반이며, 불필요한 노출을 최소화
  • 최소한의 정보만 전송 및 처리(필요 기능 수행 목적)

단, 사용자의 네트워크 환경 또는 배포 환경(Streamlit Cloud 등)에 따라 실제 적용 범위는 달라질 수 있습니다.

7. 아동(만 13세 미만) 관련

서비스는 만 13세 미만 아동을 대상으로 설계되지 않았습니다.
만 13세 미만의 이용자는 서비스를 이용하지 않도록 권고합니다.

8. 정책 변경

본 방침이 변경되는 경우, 서비스 내 공지 또는 게시를 통해 안내합니다.
변경된 방침은 게시 시점부터 효력이 발생합니다.
